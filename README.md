- `packages/api/` — FastAPI service *(Plan 2)*

- `1. VALUATION EXAMPLES/` — historical workbooks (preserved as test fixtures)
- `docs/superpowers/specs/` — design spec
- `docs/superpowers/plans/` — implementation plans
